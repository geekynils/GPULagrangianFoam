Detailed Description

Keeps all the time dependent particle related data in together. Data is stored using the vector types defined by CUDA (vector_functions.h and vector_types.h). 4 Component vectors are used because they can be fetched in one instruction (float) or in two instruction (double). The accessor functions, however return three component vector types after fetching.
